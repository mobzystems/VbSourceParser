# VbSourceParser

This console application 'parses' VB.NET source code to detect strings and comments.

Usage:

  VbSourceParser [-f] [-s] [-c] filename [filename...]

e.g.:

Some of none of the options:

-f: show file names
-s: output strings
-c: output comments

followed by at least one file name.
